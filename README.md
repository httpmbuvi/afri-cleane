# Afri Cleaning Services — Static Site

Single-file static website. No build step. Deploys instantly to **Vercel**, **Netlify**, **GitHub Pages**, **Cloudflare Pages**, or any static host.

## Deploy to Vercel
1. Push this folder to GitHub.
2. Go to https://vercel.com/new → Import the repo → Click **Deploy**. Done.

Or drag-and-drop this folder at https://vercel.com/new

## Run locally
Just open `index.html` in your browser. That's it.

## Customize
- WhatsApp number: search `254769059505` in `index.html` and replace.
- Colors: edit the `tailwind.config` block at the top of `index.html`.
- Images: replace the Unsplash URLs with your own.
